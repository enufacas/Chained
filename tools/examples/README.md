# Code Golf Optimizer Examples

This directory contains example code files that demonstrate the code golf optimizer's capabilities.

## Examples

### Python Examples
- `fibonacci.py` - Fibonacci sequence generator
- `palindrome.py` - Palindrome checker
- `prime.py` - Prime number checker

### JavaScript Examples
- `factorial.js` - Factorial calculator
- `reverse.js` - String reverser

### Bash Examples
- `backup.sh` - Simple backup script

## Usage

Optimize any example:
```bash
python3 ../code-golf-optimizer.py -f fibonacci.py -l python
```

Compare results:
```bash
python3 ../code-golf-optimizer.py -f factorial.js -l javascript --format json
```
