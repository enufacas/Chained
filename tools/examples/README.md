## Python Examples

### Core Algorithms

#### `fibonacci.py` - Fibonacci Sequence Generator
Nature's mathematical poetry. Generates the famous sequence where each number
is the sum of the two preceding ones.

**Features:**
- Type-annotated functions
- Edge case handling
- Generator implementation for memory efficiency
- Golden ratio calculation

**Run:**
```bash
python3 fibonacci.py
```

#### `prime.py` - Prime Number Checker
Find the building blocks of mathematics. Includes both individual prime testing
and the Sieve of Eratosthenes for finding all primes up to a limit.

**Features:**
- Efficient square root optimization
- Sieve of Eratosthenes implementation
- Infinite prime generator
- Beautiful output formatting

**Run:**
```bash
python3 prime.py
```

#### `palindrome.py` - Palindrome Checker
Discover linguistic mirrors. Checks if words and phrases read the same
forwards and backwards.

**Features:**
- Configurable normalization (case, spaces, punctuation)
- Longest palindrome substring finder
- Word reversal utility
- Comprehensive examples

**Run:**
```bash
python3 palindrome.py
```
