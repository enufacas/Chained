## Location Assignment Strategy

**@investigate-champion** used the following criteria for location assignments:

### 1. Tech Hub Concentration
Agents placed in cities with strong tech ecosystems:
- **Silicon Valley Area**: San Francisco, San Jose (innovation, security)
- **Seattle Area**: Seattle, Redmond (infrastructure, cloud, troubleshooting)
- **New York**: Finance and tech convergence (testing, validation)

### 2. Specialization Alignment
- **Financial Centers**: London, Frankfurt (engineering)
- **Innovation Hubs**: Tokyo, Seoul, Singapore (integration, organization)
- **Education Centers**: Boston, Copenhagen (testing, mentorship)
- **Government**: Washington DC (security monitoring)

### 3. Geographic Diversity
Ensuring global representation:
- **Americas**: North to South coverage
- **Europe**: East to West distribution
- **Asia**: All major regions represented
- **Southern Hemisphere**: Australia and Brazil included

### 4. Cultural Relevance
- **Nordic Countries**: Coaching and mentorship (known for education)
- **Asian Tigers**: Code organization (efficiency focus)
- **Silicon Valley**: Innovation and creation (startup culture)
- **European Financial**: Engineering rigor (financial system requirements)

### 5. Time Zone Distribution
Strategic spread for 24/7 coverage:
- **UTC-8 to UTC-5**: North America (10 hours)
- **UTC-0 to UTC+3**: Europe (3 hours)
- **UTC+5.5 to UTC+9**: Asia (3.5 hours)
- **UTC-3**: South America
- **UTC+10 to UTC+11**: Australia
