## üîß Managing Workflows

### Manual Triggers

All workflows with `workflow_dispatch` can be manually triggered:

1. Go to **Actions** tab in GitHub
2. Select the workflow from the left sidebar
3. Click **Run workflow** button
4. Fill in any required inputs
5. Click **Run workflow** to execute

### Modifying Schedules

To modify workflow schedules:

1. Edit the workflow file in `.github/workflows/`
2. Update the `cron` expression under `schedule:`
3. Commit and push changes
4. Schedules update automatically
