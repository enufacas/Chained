Installation

No installation required! The Code Flow Animator is a standalone Python script with no external dependencies.

git clone https://github.com/enufacas/Chained.git cd Chained/tools

Basic Usage

# Analyze a Python file python3 code-flow-animator.py -f script.py # Analyze JavaScript file python3 code-flow-animator.py -f app.js -l javascript # Generate only JSON output python3 code-flow-animator.py -f script.py --format json # Specify custom output paths python3 code-flow-animator.py -f script.py --json data.json --html report.html

Command-Line Options

  • -f, --file - Code file to analyze (required)
  • -l, --language - Programming language (auto-detected from extension if not specified)
  • --json - Output JSON file path
  • --html - Output HTML report path
  • --format - Output format: json, html, or both (default: both)
