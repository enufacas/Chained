## What is Requestly?

Requestly is a free, open-source API client and HTTP interceptor that serves as an alternative to Postman, Charles Proxy, Fiddler, and Insomnia. It combines API testing capabilities with powerful request interception features.

### üîÑ HTTP Interception

- Change headers, query parameters, request/response bodies
- Delay or block requests for testing
- Redirect requests dynamically
- Works without VPN or complex proxy setup

### üöÄ API Client
- Send and manage API requests
- Organize requests in collections
- Manage multiple environments
- JavaScript scripting for pre/post-request logic
- Support for all major auth methods (OAuth, Bearer, API Keys, Basic Auth)
