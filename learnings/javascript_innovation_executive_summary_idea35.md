### Want to Try Deno? (5 minutes)

```bash
# Install Deno
curl -fsSL https://deno.land/install.sh | sh

# Create test script
cat > test.ts << 'EOF'
// Deno example with TypeScript
const response = await fetch("https://api.github.com/users/deno");
const data = await response.json();
console.log(`Deno has ${data.public_repos} public repos`);
EOF

# Run with permissions
deno run --allow-net test.ts

# Run without permissions (will fail)
deno run test.ts  # Shows permission error!
```

### Want to Try Edge Deployment? (10 minutes)

```bash
# Install Wrangler (Cloudflare CLI)
npm install -g wrangler

# Create edge function
npx wrangler init my-worker

# Write simple function
cat > src/index.ts << 'EOF'
export default {
  async fetch(request: Request): Promise<Response> {
    return new Response("Hello from the edge!");
  },
};
EOF

# Deploy globally
npx wrangler deploy
```
